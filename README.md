Black-Scholes Option Pricing + Monte Carlo Risk Engine

This project implements two mathematical approaches to European option pricing and financial risk modelling:

    Black-Scholes Analytical Pricer — closed-form solution derived from a PDE using Ito's lemma
    Monte Carlo Simulation — numerical method using Geometric Brownian Motion (100,000 paths)
    Greeks Calculator — partial derivatives of the option price (Delta, Gamma, Theta, Vega)
    Real Market Validation — applied to Reliance Industries (NSE) using RBI repo rate and historically computed volatility
